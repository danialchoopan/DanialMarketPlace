# پروژه MarketPlaceCore

مدیریت هسته مرکزی فروشگاه اینترنتی بر پایه ASP.NET Core با معماری Clean Architecture و الگوهای Repository و Unit of Work.

## ساختار پروژه

پروژه به سه لایه اصلی تقسیم شده است:
۱. **MarketPlaceCore.Core**: شامل موجودیت‌ها (Entities) و اینترفیس‌های اصلی.
۲. **MarketPlaceCore.Infrastructure**: پیاده‌سازی دسترسی به داده‌ها با EF Core، الگوهای مخزن و مدیریت تراکنش‌ها.
۳. **MarketPlaceCore.Web**: شامل APIها، صفحات Razor و رابط کاربری با Tailwind CSS.

## ویژگی‌های کلیدی
- جداسازی لایه‌ها برای تست‌پذیری بالا.
- مدیریت یکپارچه تراکنش‌ها (Unit of Work) جهت تضمین پایداری داده‌ها در ثبت سفارش.
- پشتیبانی از تم تاریک و روشن به صورت محلی.
- سیستم بذرپاشی داده‌ها (Database Seeding) برای آماده‌سازی سریع محیط.

## راهنمای راه‌اندازی

۱. ابتدا وارد پوشه اصلی پروژه شوید.
۲. با دستور زیر وابستگی‌ها را نصب و پروژه را اجرا کنید:

```bash
dotnet run --project src/MarketPlaceCore.Web/MarketPlaceCore.Web.csproj
```

دیتابیس به صورت خودکار در اولین اجرا ساخته شده و با داده‌های نمونه پر می‌شود.
