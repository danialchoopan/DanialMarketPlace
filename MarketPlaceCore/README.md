# پروژه MarketPlaceCore (نسخه اینترپرایز)

هسته جامع و پیشرفته فروشگاه اینترنتی بر پایه ASP.NET Core با معماری Clean Architecture و الگوهای Repository و Unit of Work.

## ویژگی های کلیدی و پیشرفته

### ۱. سیستم تنوع محصول و انبارداری (Variants)
مدیریت هوشمند محصولات با چندین تنوع (رنگ، سایز، گارانتی) که هر کدام دارای قیمت و موجودی انبار مستقل هستند.

### ۲. موتور تخفیف هوشمند (Discount Engine)
اعمال کدهای تخفیف با شروط تاریخ انقضا، سقف استفاده و حداقل مبلغ خرید در لایه تراکنش های پایگاه داده.

### ۳. نظرات و رتبه بندی خریداران واقعی
سیستم ثبت نظر با قابلیت تشخیص خودکار خریداران (Verified Buyer) بر اساس سوابق سفارشات تحویل داده شده.

### ۴. احراز هویت چندگانه
پشتیبانی از ورود با نام کاربری/رمز عبور و همچنین ورود سریع با کد یکبار مصرف (OTP) شبیه سازی شده.

### ۵. لیست علاقه مندی ها و اطلاع رسانی
مدیریت لیست های شخصی کاربران و قابلیت ثبت درخواست اطلاع رسانی برای کالاهای ناموجود.

## ساختار لایه بندی
- **Core**: شامل موجودیت های پیشرفته، Enumها و اینترفیس های مخازن.
- **Infrastructure**: پیاده سازی EF Core، مدیریت تراکنش های Unit of Work و امنیت (SHA256).
- **Web**: شامل APIهای DTO-based و رابط کاربری مدرن با Tailwind CSS بدون وابستگی به CDN.

## راهنمای راه اندازی سریع

۱. اجرای پروژه:
```bash
dotnet run --project src/MarketPlaceCore.Web/MarketPlaceCore.Web.csproj
```
دیتابیس در اولین اجرا ساخته شده و با داده های حجیم شامل تنوع ها، نظرات و کدهای تخفیف بذرپاشی می شود.
